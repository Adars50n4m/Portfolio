Since this project is currently running on `localhost`, it is only visible on your computer. To share it with others, you need to deploy it.

### Option 1: Vercel (Recommended)
1.  Create a [Vercel account](https://vercel.com).
2.  Install Vercel CLI: `npm i -g vercel`
3.  Run `vercel login`
4.  Run `vercel` in this project folder.
5.  Follow the prompts (hit Enter for defaults).
6.  **Done!** You will get a link distinct like `https://adarsh-portfolio.vercel.app`.

### Option 2: Netlify
1.  Create a [Netlify account](https://netlify.com).
2.  Drag and drop the `dist` folder (run `npm run build` first to create it) into the Netlify dashboard.
